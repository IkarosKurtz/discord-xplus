    LevelManager

    Extends
    • EventEmitter
    Creates a LevelManager, this is the main class, when you create a new instance of this class, it will automatically connect to the database and load the users (if you already used the manager), also will load the events if you specified the path to the events folder in the options, but if you don't have a folder or you doesn't know how it works, you can visit Events and eventsPath to learn how to create and use events.
