    Type Definitions

    LevelManagerOptions

    Type

    • Object

    Properties

    • client Discord.Client
      The Discord client
    • mongoURI string
      The URI to connect to the MongoDB database
    • maxXpToLevelUp number <optional>
       2500
      The maximum XP to level up (default: 2500)
    • logChannelId string <optional>
      The ID of the channel to log the events
    • eventsPath string <optional>
      The path to the events folder (example: ./events/LevelEvents)
    • ranks Array.<(Rank|RankBuilder)> <optional>
       Ranks
      The ranks of the server (default: Ranks)
    • autosave boolean <optional>
       true
      If the manager should autosave the data (default: true)

    Example

    const Ranks = [
	{
		nameplate: 'Noobie',
		color: 'Red',
		min: 0,
		max: 9,
		value: 0
	},
	{
		nameplate: 'Rookie',
		color: 'Orange',
		min: 10,
		max: 19,
		value: 1
	}
]
