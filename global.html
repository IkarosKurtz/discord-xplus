    Type Definitions

    LevelManagerOptions

    Options that will be passed to the manager.

    Type

    • Object

    Properties

    • mongoURI string
      The URI to connect to the MongoDB database.
    • maxXpToLevelUp number <optional>
       1500
      The maximum XP to level up.
    • logChannelId Discord.Snowflake <optional>
      The ID of the channel to log the events.
    • eventsPath string <optional>
      The path to the events folder (example: ./events/LevelEvents).
    • ranks Array.<(Rank|RankBuilder)> <optional>
       Ranks
      The ranks of the server.
    • autosave boolean <optional>
       true
      If the manager should autosave the data.
    • calculateXpFunction xpFunction <optional>
       (level, xp) => level * 2500
      The function to calculate the XP.
    • msToSave number <optional>
       1000 * 60 * 60 * 4
      The time in milliseconds to save the data.

    xpFunction(level, xp) → {number}

    Callback function to calculate the XP to level up.

    Parameters

    • level number
      The current level of the user.
    • xp number
      The current XP of the user (if user has level up, this will be the XP after the level up).

    Returns

    • number The necessary XP to level up.
